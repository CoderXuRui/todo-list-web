# 待办应用产品规划

## 1. 产品定位

**目标用户**：需要规划日常任务的人群

**核心价值**：一款极简、可爱风格的本地待办应用

**灵感来源**：Todoist (https://app.todoist.com/app/inbox)

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 任务管理 | 添加、修改、删除任务 | 必须 |
| 任务列表 | 支持按日期、分类、优先级查询/排序 | 必须 |
| 分类筛选 | 用户可创建自定义分类（工作、学习等） | 必须 |
| 每日统计 | 展示总任务数、已完成数、完成率 | 必须 |

### 2.2 统计面板

- 总任务数
- 已完成任务数
- 完成率

### 2.3 分类管理

- 用户自定义分类名称
- 分类颜色/图标（可选）
- 支持创建、编辑、删除分类

### 2.4 任务属性

- 标题
- 分类
- 优先级（高/中/低）
- 截止日期
- 完成状态

---

## 3. 技术要求

- 使用 LocalStorage 存储数据（容量 5MB）
- 每次操作后自动保存，无需手动保存
- 支持导出/导入 JSON 文件作为备份

---

## 4. 数据结构设计

```json
{
  "tasks": [
    {
      "id": "uuid",
      "title": "任务标题",
      "category": "工作",
      "priority": "high|medium|low",
      "deadline": "2026-04-22",
      "completed": false,
      "createdAt": "2026-04-22T10:00:00Z"
    }
  ],
  "categories": [
    { "id": "uuid", "name": "工作", "color": "#FF6B6B" },
    { "id": "uuid", "name": "学习", "color": "#4ECDC4" }
  ],
  "settings": {
    "lastUpdated": "2026-04-22T10:00:00Z"
  }
}
```

### 4.1 存储 Key 规划

| Key | 内容 |
|-----|------|
| `todo_tasks` | 任务列表 |
| `todo_categories` | 分类列表 |
| `todo_settings` | 设置信息 |

### 4.2 防丢失措施

- 每次操作后立即保存到 LocalStorage
- 支持手动导出 JSON 备份
- 启动时检查数据完整性
